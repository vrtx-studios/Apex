# Apex - MVC-framework
## What is Apex?
Apex is a platform meant to build specific web-applications.
### What does that mean?
It means if you need a purpose-built E-Commerce, a CRM, PIM, etc., you can use Apex to do the heavy-lifting for you.

## Cool beans, is it useable?
Well, Apex is still undergoing the initial development, but you can always download the code and make the rest for yourself if you're too excited.
